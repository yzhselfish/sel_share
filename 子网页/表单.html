简介

一个完整的表单通常由表单域、表单控件（也称表单元素）和提示信息3个部分构成。

表单的标签为form，是一个双标签。里面有三个属性action、method和name

  • action=“url地址”指定接受并处理表单数据的服务器程序的url
  • method=“提交方式”设置提交方式，一般为post和get
  • name=“表单域名称”指定表单名称，区分同一个页面的多个表单域

input标签

input是一个单标签，包含一个type属性，根据不同的type属性值，输入字段有很多形式

  • button 定义可点击按钮，不是提交数据，用于JS启动脚本
  • checkbox 定义复选框
  • file 定义输入字段和“浏览”按钮，供文件上传
  • hidden 定义隐藏的输入字段
  • image 定义图像形式的提交按钮
  • password 定义密码字段，字符被掩码
  • radio 定义单选按钮，他们的name属性必须相同才能多选一
  • reset 定义重置按钮，会清除表单中的所有数据，返回默认状态
  • submit 定义提交按钮，会把表单数据发送到服务器，通过value改变名称
  • text 定义单行的输入字段，用户可在其中输入文本，默认宽度为20个字符。

input除了type属性，还有其他属性，常用属性如下

  • name属性定义input元素的名称，单选和复选框要有相同的name
  • value规定input元素的默认值，文本框可以显示，按钮不显示
  • checked规定此input元素首次加载时应当被选中，主要用于单选和复选框
  • maxlength规定输入字段中的字符最大长度

label标签

label并不属于表单标签，而是为input定义标注，是一个双标签，把文字放在label标签里面

label绑定一个表单元素，当点击label标签内的文本时，浏览器会自动将光标转到或者选择对应的表单元素，用来增加用户体验

label标签有一个for属性，与相关元素的id属性相同完成绑定（注意：这里不用加锚点链接里面的#）

select标签

如果有多个选项让用户选择，并且想要节约空间，可以用select标签定义下拉列表

select是一个双标签，把其中的选项放在option标签里面

在option里面定义selected属性，selected="selected"，当前项即为默认选中选

textarea标签

当用户输入内容较多的情况下，不能使用文本框表单了（只能显示一行），可以使用textarea标签，是用于定义多行文本输入的一个控件

里面有两个属性，cols=“每行的字符数”，rows=“显示的行数”，实际开发一般用CSS改变大小
