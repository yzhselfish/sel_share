网页布局

网页布局的过程如下：

  1. 先准备好相关的网页元素，网页元素基本都是盒子box
  2. 利用CSS设置好盒子样式，然后摆放到相应位置（核心本质）
  3. 往盒子里面装内容

盒子模型(Box Model)组成

盒子模型就是把HTML页面中的布局元素看作是一个矩形的盒子，也就是一个盛装内容的容器

CSS盒子模型封装周围的HTML元素，包括：边框、外边距、内边距和实际内容

border边框 content内容 padding内边距 margin外边距

边框(border)

border可以设置元素的边框，边框由三部分组成：边框宽度（粗细） 边框样式 边框颜色

border-width定义边框粗细，单位是px border-style是边框的样式，solid实线边框dashed虚线边框dotted点线边框border-color边框颜色

边框的简写border: 1px solid red;没有顺序

边框还可以分开写，四条边分别设置，border-top: xxpx solid color;只设定上边框

border-collapse: collapse;合并相邻的边框

边框会影响盒子实际大小

内边距

padding属性用于设置内边距，即边框与内容之间的距离，padding-left左内边距 其他边距依次类推

padding也可以简写，依次控制上、右、下、左 顺时针，只写一个值使上下左右都有相应的边距

写两个值前一个代表上下边距，后一个代表左右边距，三个值第一个代表上边距，第二个左右边距，第三个是下边距，四个值就是上右下左

如果遇到盒子里面的字数不相等我们可以不固定设置盒子的宽度，而是给盒子设置padding更加合理，padding将盒子撑开

如果盒子本身没有指定width/height属性，则padding不会撑开盒子大小

外边距(margin)

margin用于控制盒子和盒子之间的距离，margin-left左外边距，其他的依此类推，简写方式与padding一致，都是上右下左

外边距可以让块级盒子水平居中，但是盒子首先必须指定了宽度，盒子左右的外边距都设置为auto

如果想要让盒子里面的行内元素或者行内块元素水平居中给父级加上text-align: center;就可以了

外边距合并问题

使用margin定义块元素的垂直外边距时，可能会出现外边距的合并，主要有两种情况

1.相邻块元素垂直外边距的合并

上下相邻的两个块元素（兄弟关系）相遇时，他们之间的垂直边距不是两个边距的和，而是其中较大的一个

解决方法：尽量只给一个盒子添加margin值。

2. 嵌套块元素垂直外边距的塌陷

对于两个嵌套关系（父子关系）的块元素，父元素有上外边距同时子元素也有上外边距，此时父元素会塌陷较大的外边距值

解决方法

  1. 可以为父元素定义上边框
  2. 可以为父元素定义上内边距
  3. 可以为父元素添加overflow:hidden

注意：浮动、固定，绝对定位的盒子不会有塌陷问题

清除内外边距

网页元素很多都带有默认的内外边距，而且不同浏览器默认的也不一致，因此我们在布局前，首先要清除下网页元素的内外边距

解决方法就是写下下面的代码*{margin: 0;padding: 0;}，*为通配符选择器

行内元素为了照顾兼容性，尽量只设置左右内外边距，不要设置上下内外边距，如果一定要设置上下边距可以转换为块级和行内块元素

PS基础

快捷键Ctrl+R调出标尺，也可以通过视图菜单里面的标尺，默认单位是cm，但是网页设计我们常用px像素作为单位，右击标尺把里面的厘米改成像素

Ctrl+加号可以放大视图，Ctrl+减号可以缩小视图，按住空格可以移动图片

我们用选区拖动测量大小，Ctrl+D可以取消选区，或者在旁边空白处点击一下也可以取消选区，用吸管工具进行取色

常见的图片格式

jpg:JPEG(JPG)对色彩的信息保留较好，高清颜色多，产品类的图片经常用jpg格式

gif:GIF格式最多只能储存256色，所以通常用来显示简单图形及字体，但是可以保存透明背景和动画效果，实际经常用于一些图片小动画特效

png:是一种新兴的网络图形格式，结合了GIF和JPEG的优点，具有存储形式丰富的特点，能够保存透明背景，如果想要切成背景透明的图片，选择png

PSD:是Photoshop的专用格式，里面可以存放图层、通道、遮罩等多种设计稿，对前端人员来说，最大的优点是可以直接复制文字，获得图片，还可以测量大小和距离

PS切图

PS有很多的切图方式：图层切图、切片切图、PS插件切图等

图层切图：右击图层->快速导出为png，有很多时候需要合并图层再导出，选中需要的图层：图层菜单->合并图层(ctrl+e)->右击->快速导出为png

切片切图：利用切片工具手动划出，文件菜单->导出->存储为web设备所用格式->选择我们要的图片格式->存储

插件切图：Cutterman是一款运行在PS中的插件，能够自动将你需要的图层进行输出，以替代传统的手工导出以及使用切片工具进行挨个切图的繁琐流程

圆角边框

border-radius:length;属性用于设置元素的外边框圆角，radius（半径）

除了以px为单位，还可以用百分比的形式，比如50%就可以做一个正圆形

该属性是一个简写属性，可以跟四个值，分别代表左上角、右上角、右下角、左下角，两个值前一个是左上和右下，后一个设置的是右上和左下

也可以分别设置，比如设置左上角可以写作border-top-left-radius

盒子阴影

我们用box-shadow这个属性为盒子添加阴影，语法box-shadow: h-shadow v-shadow blur spread color inset，默认outset是不能写的

属性值 描述
h-shadow 必选，水平阴影的位置，可负
v-shadow 必选，垂直阴影的位置，可负
blur 可选，模糊距离
spread 可选，阴影的尺寸
color 可选，阴影的颜色
inset 可选将外部阴影改为内部阴影

盒子阴影不占用空间，不会影响其他盒子的排列，如果想让鼠标放在上面再出现影子可以用hover

文字阴影

用text-shadow这个属性将阴影用于文本，语法box-shadow: h-shadow v-shadow blur color

属性值 描述
h-shadow 必选，水平阴影的位置，可负
v-shadow 必选，垂直阴影的位置，可负
blur 可选，模糊距离
color 可选，阴影的颜色
